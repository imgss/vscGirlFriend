# vscGirlfriend README

一个在你长时间写代码时提醒你多喝热水的vscode插件

## Features

1. 提醒你多喝热水
1. 提醒你左三圈，右三圈，脖子扭扭，屁股扭扭
1. 提醒你起来走动走动
1. 提醒你多喝热水...
